LiberaEventReceiver Class Commands
Name
Input type
Output type
Level
Description
State DEV_VOID DEV_STATE OPERATOR This command gets the device state (stored in its device_state data member) and returns it to the caller.
Status DEV_VOID CONST_DEV_STRING OPERATOR This command gets the device status (stored in its device_status data member) and returns it to the caller.
GetParameters DEV_VOID DEVVAR_DOUBLESTRINGARRAY OPERATOR Retruns the current gains and offsets (i.e. the parameters used for position computation)
ReloadSystemProperties DEV_VOID DEV_VOID OPERATOR None.
MagicCommand DEV_STRING DEVVAR_STRINGARRAY EXPERT Directly dump the Libera ireg node structure. If the input argument is a sub tree all subnodes will be dumped out.
AnnounceSynchronization DEV_VOID DEV_VOID OPERATOR Announce synchronization procedure:
* Stop the Trigger (t2source=off)
* MC PLL (MUST be locked)
* Synchronization State Machine (MUST be Tracking)
ForceInitSettings DEV_VOID DEV_VOID OPERATOR Initializing Libera Default Settings (Based to the properties) without performing init_device
SetTraceLevel DEV_USHORT DEV_VOID OPERATOR Sets the Trace Error Level: Off = 0,
Low = 1,
Med = 2,
High = 3,
Detail = 4
StartSynchronization DEV_VOID DEV_VOID OPERATOR Starts synchronization procedure:
* Synchronization State Machine (MUST be Tracking)
* Start the Trigger (t2source=Pulse and then t2source=RTC)
* Synchronization State Machine (MUST be Synchronized)
