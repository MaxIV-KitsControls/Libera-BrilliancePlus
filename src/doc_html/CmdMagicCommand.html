Command MagicCommand :
    Directly dump the Libera ireg node structure. If the input argument is a sub tree all subnodes will be dumped out.



MagicCommand Definition
Input Argument Tango::DEV_STRING absolute path of the node name
Output Argument Tango::DEVVAR_STRINGARRAY value of specified node(s)
